1. The user is able to add a due date to their tasks using the "set_due_date" method, the default value of due date is "Today". When you add a task, the time and date is taken as an attribute using "Time.now"
2. Using the method "save_to_file" the user can save their todo list to a file called "todo.txt".
3. Added priority to tasks, if it is set as important, it will print with a star, the default value for each item is "Low", using the method "set_high_importance" it can be changed.
